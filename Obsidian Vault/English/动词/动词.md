# 动词的定义

表示动作或状态的词
# 动词的分类

![[动词的分类.excalidraw.svg]]
## 实义动词

有实际意义的动词
有完整词义，能够单独作谓语

实义动词分为二类

````col
```col-md

### 一，及物动词：vt
后面可以直接且必须涉及到人或物


```

```col-md

eg：hit(打)，I hit
hit后面什么都不跟，”我打“，打什么？
会显得句子表达的不完整




```
````

````col
```col-md

### 二，不及物动词：vi
后面不需要涉及到人或物
(后面想要涉及人或物时需要+介词)

```

```col-md

eg：come(来)
I come 我来了
句子后面没跟人或物，就已经表达完整了
I come from China (我来自中国)



```
````
## 系动词

后面跟n/adj构成系表结构
主要分为两类

````col
```col-md

### 一，be类系动词：
am，is，are

```

```col-md

eg：
She is a $\underline {teacher.(n.)}$
She is <u>beautiful.(adj.)</u>


```
````

````col
```col-md

### 二，感官类系动词：
和人的器官有关系
sound，look，smell，feel，taste

```

```col-md

eg：
The food taste $\underline {wonderful.(adj.)}$


```
````
## 情态动词
情：情感
态：态度
有一定的词义，后面必须要跟动词原形，与动词原形一起构成谓语
常见的有：

````col
```col-md

should = ought to，could，can，must，might，may，need，dare，had，better，have to，used to

```

```col-md

could
是can更委婉的说法
是can的过去式



```
````
### 情态动词无人称和数的变化

````col
```col-md

不管是第一人称还是第三人称都用原型
不管是单数还是复数都用原型
注：have to除外

```

```col-md

eg：
I <u>shuold</u> study hard.
She <u>should</u> study hard.



```
````
### 含有情态动词的句子变否定
````col
```col-md

直接在情态动词后面+not
注：have to除外

```

```col-md

eg：
we $\underline {should}$ clean the room.
we $\underline {shouldn't}$ clean the room.



```
````
### dare/need
````col
```col-md

既可以做情态动词，又可以做实义动词

```

```col-md

eg：
He <u>need learn</u> the song.(情态动词)
He <u>needs to go</u> home.(实义动词)
He <u>daren't speak</u> English before many people.(情态动词)
Tom <u>dares to do</u> the thing.(实义动词)



```
````
## 助动词
助动词不可单独使用，
通俗的说，助动词就是辅助动词的，一般没有实际意义
助动词分为四大类（do类，be类，have类，will类）
### do类助动词
````col
```col-md

do(原形)，does(单数)，did(过去式)
用于帮助句子构成否定和疑问

```

```col-md

eg:
I like you.(我喜欢你)
I do not like you.(我不喜欢你)
Do i like you?(我喜欢你吗？)



```
````
### be类助动词
am，is，are，was，were
用于帮助动词构成进行时或被动语态
### have类助动词
have，has，had
用于帮助动词构成完成时
### will类助动词
will，would，shall
用于帮助动词构成将来时

## 用原形/单数的情况
用原形还是单数取决于主语，因为英语要主谓一致
### 当主语是人称代词时
````col
```col-md

第一人称，单数(I)，复数(We)，作主语
谓语用动词原形

```

```col-md

eg:
I come from China.
We come from China.



```
````

````col
```col-md

第二人称，单数(You)，复数(You)，作主语
谓语用动词原形

```

```col-md

eg:
You come from China.



```
````

````col
```col-md

第三人称，复数(They)，作主语
谓语用动词原形

```

```col-md

eg:
They come from China.



```
````

````col
```col-md

第三人称，单数(He，She)，作主语
谓语用动词==单数==

```

```col-md

eg:
<u>She comes</u> from China.



```
````

总结：第三人称单数作主语，谓语动词用==单数==，其余用原形
### 当主语是名词时
````col
```col-md

可数名词单数作主语，后面的动词用单数

```

```col-md

eg:
This apple tastes good.



```
````

````col
```col-md

不可数名词作主语，后面的动词用单数

```

```col-md

eg:
Bread tastes good.


```
````

````col
```col-md

可数名词复数作主语，后面的动词用==原形==

```

```col-md

eg:
<u>These apples taste</u> good.



```
````

总结：可数名词复数作主语，后面的动词用==原形==，其余用单数
# 动词的变化规则
主要分为三种
==注：==辅音字母：除元音字母(a，e，i，o，u)之外的字母
## 动词原形 $\Longrightarrow$ 动词单数
````col
```col-md
==一般情况==
直接在词尾加s
eg：
play $\to$ plays
say $\to$ says
work $\to$ works
```

```col-md

==辅音字母加y结尾==
y变i加es
eg：
study(d+y) $\to$ studies
try(r+y) $\to$ tries
fly(l+y) $\to$ flies

```

```col-md

==以s，x，ch，sh，o结尾==
直接加es
eg：
pass $\to$ passes
finish $\to$ finishes
fix $\to$ fixes
wish $\to$ wishes
do $\to$ does


```
````

## 动词原形 $\Longrightarrow$ 动词过去式/过去分词

````col
```col-md
==一般情况==
直接在词尾加ed
eg：
work $\to$ worked
watch $\to$ watched
```

```col-md

==以e结尾==
直接加d
eg：
live $\to$ lived
like $\to$ liked
die $\to$ died

```
````

````col
```col-md
==辅音字母加y结尾==
y变i加ed
eg：
cry(r+y) $\to$ cried
study(d+y) $\to$ studied
```

```col-md

==由辅元辅结尾==
双写尾字母加ed
eg：
stop(t+o+p) $\to$ stopped
plan(l+a+n) $\to$ planned
==特==：
open(p+e+n) $\to$ opened
```
````
==不规则动词变化表==
## 动词原形 $\Longrightarrow$ 动词ing
````col
```col-md
==一般情况==
直接在词尾加ing
eg：
work $\to$ working
do $\to$ doing
play $\to$ playing
see $\to$ seeing
```

```col-md

==以不发音字母e结尾==
去e加ing
eg：
dance $\to$ dancing
move $\to$ moving
decide $\to$ deciding
大多数词尾的e是不发音的
少数发音的：see

```
````
````col
```col-md
==由辅元辅结尾==
双写尾字母加ing
eg：
get(g+e+t) $\to$ getting
shop(h+o+p) $\to$ shopping
run(r+u+n) $\to$ running
swim(w+i+m) $\to$ swimming
```

```col-md

==以ie结尾==
ie变y加ing
eg：
lie $\to$ lying
die $\to$ dying

```
````
