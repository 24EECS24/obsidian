# 算法及其性能分析

概念：解决问题的操作
特性：有限性，确定性，可行性，输入，输出
性能指标o(n)：时间复杂度，空间复杂度
空间复杂度不常考
时间复杂度看循环，（时间复杂度越小，执行时间越短，性能越高）


````col

```col-md
```c
#include <stdio.h>
int main(){
	int j,i,n;
	for(i=0;i<n;i++){
		for(j=0;j<n;j++){
			printf("*");
		}
	}		
}
```


```col-md 
$$单个for循环是执行了n次，$$
$$将n看成n\longrightarrow \infty $$
$$又是嵌套的循环$$
$$\therefore 是执行了n\times n= n^{2} 次$$
$$\therefore 时间复杂度为o(n^{2} )$$
````
